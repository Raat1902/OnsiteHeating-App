# Dev Notes (v1 Alpha)

This repo is a portfolio demo. My goal for v1 is a clean end-to-end workflow (dispatch → job → invoice) with local persistence.

## Quick checks I run
```bash
npm install
npm run build
npm run preview
```

## Notes
- Data is stored in the browser (IndexedDB) for the demo.
- There is no backend in v1; authentication is demo-only.
